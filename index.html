hc by houbysoft

An open source calculator with a simple and easy-to-use interface, similar to a command prompt, but a lot of options and a multitude of functions.

An open source calculator with a simple and easy-to-use interface, similar to a command prompt, but a lot of options and a multitude of functions. More information is available at http://houbysoft.com/hc/

Dependencies

libmapm libreadline

Install

Use make to compile the program; then the binary can be launched with ./hc. For archlinux users, packages are provided in the AUR.
